Gdscript Transpiler

Gdscript 2.0 updater.

The upcoming Godot 4 release bears many changes to both the core works of the game engine and the scripting language, Gdscript. Subsequently, this tool has been created in order to automatically update "1.0" scripts using the outdated godot 3x API, to "2.0" scripts, allowing them to be used with godot 4.0 projects.
